# SafetyExplosions

A simple Minecraft plugin that protects blocks from all types of explosions. Supports Folia

```yaml

prevent-blocks-from-exploding:
  end_crystal: true
  respawn_anchor/bed: true
  tnt: true
  tnt_minecart: true
  creeper: true

invulnerable-entities:
  item_frame: true
  armor_stand: true

```
